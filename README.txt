== Description ==

Simplistic WordPress theme for documentation. A guide for your guide.

For more information about Squire please go to https://github.com/sixteenbit/squire.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click on Add New -> Upload Theme.
3. Choose the zip and upload the theme.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
